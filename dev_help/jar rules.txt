.jar est créé par Eclipse
===========================
Un fichier JAR (Java archive) est un fichier ZIP utilisé pour distribuer un ensemble de classes Java. 
Ce format est utilisé pour stocker les définitions des classes, ainsi que des métadonnées, constituant l'ensemble d'un programme.
Attention à la notion de ressource.
 A resource in a jar file is not a File, so you can't treat it as one.

Un jar a besoin d'une JVM
Trouvé à travers les variables d'environnement de Windows
Attention par contre, le fichier .jar à besoin du répertoire ressources, qui dans le programme java a été spécifié en relative
		this.fileNameTP = "./ressources/R22/P2T_R22.txt";

		
Launch4j
========
Launch4j est un logiciel qui permet de créer des lanceurs .exe pour vos applications Java.
Il va permettre de dire à .java ou se trouve le jre.
Mais comme pour le .jar il faut spécifier les répertoires
\jrex86
\jrex64
\ressources
\log
PAC-Toolx64.exe
PAC-Toolx86exe

Installation 
==============
Nous allons en fait utiliser inno pour packager tout cela de telle sorte qu'au final, on se retrouve avec executable qui
va installer tous les répertoires
